# 🌍 EcoNexus

### Smart Waste Platform — Waste Intelligence & Predictive Eco-Infrastructure

EcoNexus is a full-stack predictive waste management platform designed to help municipal authorities, city administrators, and large organizations optimize solid waste systems. By pairing a modern data dashboard with advanced AI architectures, the platform moves away from reactive waste management and transitions into data-driven, predictive eco-governance aligned with the United Nations Sustainable Development Goals (SDG 11 & SDG 12).

---

# 📋 Problem Statement

Traditional urban waste management systems remain largely reactive, creating operational inefficiencies, increased costs, and environmental strain.

Many municipalities struggle with:

## 🚮 Lack of Real-Time Logistics Data

Difficulty identifying localized waste accumulation spikes before bins overflow and collection systems become overwhelmed.

## 📈 Unpredictable Waste Generation

Waste volumes often surge during holidays, public events, seasonal changes, and institutional activities, resulting in poor resource allocation.

## 📑 Limited Sustainability Transparency

Organizations frequently lack clear, verifiable mechanisms for tracking waste-to-resource conversion, compliance monitoring, and environmental reporting.

---

# ✨ Solution

EcoNexus transforms waste management from a reactive process into a predictive intelligence system.

By combining localized data collection, GIS-based spatial analytics, AI forecasting pipelines, and sustainability auditing tools, the platform enables city planners and institutional administrators to:

* Predict waste generation before it occurs.
* Optimize collection schedules and fleet deployment.
* Identify high-risk waste accumulation zones.
* Monitor sustainability performance.
* Generate transparent compliance reports.
* Support data-driven environmental governance.

# 🚀 Installation & Setup

## Prerequisites

Ensure the following are installed:

* Node.js (v18+ recommended)
* npm

Verify installation:

```bash
node -v
npm -v
```

---

## Clone Repository

```bash
git clone https://github.com/nilaninil/Econexus.git

cd Econexus
```

---

## Install Dependencies

```bash
npm install
```

---

## Configure Environment Variables

Create a `.env` file from the example template.

### Mac/Linux

```bash
cp .env.example .env
```

### Windows

```bash
copy .env.example .env
```

Add the following values:

```env
VITE_GEMINI_API_KEY=your_google_gemini_api_key_here

IBM_GRANITE_API_KEY=your_ibm_granite_api_key_here

PORT=5000
```

---

## Run Development Server

```bash
npm run dev
```
